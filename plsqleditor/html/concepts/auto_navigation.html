Auto Navigation

You can execute file lookup of any function call, procedure call etc that is selected in the editor. This lookup will navigate to the file containing the object you have selected and then to the location in the file defining that object.
Lookup can be executed by the following mechanisms:

  • F3 (while in Toby's PLSQL Editor Perspective) on:
    • packages
    • package.procedure
    • package.field
    • package.function
    • local fields
    • parameters
