PlSQLGenHeader

Description

Generates the headers of any supplied PLSQL Bodies

Parameters

Attribute Description Required
plsqlfile The Source PLSQL Body file to generate the header from. Yes, unless a nested resource collection element is used.
destfile The Destination PLSQL Header file to create No; defaults to false.
headerextention The extention to give all generated header files No; defaults to .pkh

Parameters specified as nested elements

fileset

A fileset or multiple file sets can be added to include a set of files to generate headers from. The Headers will be created with either the supplied headerextention option or defaulted to .pkh
The headers will have the name of the supplied source body.

Examples

taskdef

You need to define this library prior to using it. This can be done by including the following at the head of your build xml.

  <taskdef name="plsqlheadgen" classname="plsqleditor.anttasks.PlsqlHeaderGenTask">
     <classpath>
   	  <fileset dir="./lib">
   	    <include name="*.jar"/>
   	  </fileset>
    </classpath>
  </taskdef>

Generate a single header

  <plsqlheadgen plsqlfile="myfile.pkb" destfile="myfile.pkb"/>

If the destfile is left blank, the filename is assumed to be the same as the plsqlfile with pkh as an extension instead of pkb.

Generate a directory of bodies

  <plsqlheadgen>
    <fileset dir="${build-dir}/database" includes="**/*.pkb"/>
  </plsqlheadgen>

This will include all pkb files underneath build_dir/database.
