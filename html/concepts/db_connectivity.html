Oracle Driver

This is only global in the preferences

Connect string (aka Url)

This URL should be in the form jdbc:oracle:thin:@host_name:1521:SID

User name

This must be a db admin type user who has access to all the schemas you wish to access for auto lookup.

Password

This is the password of the aforementioned user

Initial Connections

This can be one

Max Connections

Set this to three or more - one is not enough
