packages

Packages are one technique that Oracle uses to group executable functionality.
Packages can contain types, objects, constants, cursors, procedures and functions.
They do not contain tables.
This is the primary type of file that the plsql editor supports. You can edit any file that has sql components in it, but at the moment the most strength comes from package files.
